# Dataset Access Instructions

## Source

This study uses multi-omics data from The Cancer Genome Atlas (TCGA) accessed through the MLOmics repository.

## MLOmics Repository

- **Paper:** Chen Z, Luo Z, Li Y, et al. "Cancer Multi-Omics Benchmark (MLOmics)."
- **GitHub:** [https://github.com/chenzRG/Cancer-Multi-Omics-Benchmark](https://github.com/chenzRG/Cancer-Multi-Omics-Benchmark)
- **Direct data download:** [https://figshare.com/](https://figshare.com/) (search for MLOmics)
- **Hugging Face mirror:** [https://huggingface.co/datasets/chenzRG/MLOmics](https://huggingface.co/datasets/chenzRG/MLOmics)

## Cancer Types Used (12 total)

ACC, COAD, GBM, KIRC, KIRP, LGG, LIHC, LUSC, OV, PRAD, THCA, THYM

## Omics Modalities (4 total)

- **mRNA expression** -- RNA-seq, 25,024 genes
- **miRNA expression** -- miRNA-seq, 2,062 miRNAs
- **DNA methylation** -- Illumina 450K, 25,545 CpG sites
- **Copy number variation** -- SNP6 arrays, 21,482 segments

## Download Instructions

1. Clone or download the MLOmics repository from GitHub:
   ```bash
   git clone https://github.com/chenzRG/Cancer-Multi-Omics-Benchmark.git
   ```

2. Place the `Clustering_datasets` and `Classification_datasets` folders under `data/MLOmics/Main_Dataset/`.

3. The notebook expects the following data structure:
   ```
   data/MLOmics/Main_Dataset/Clustering_datasets/{CANCER_TYPE}/Original/
   ```
