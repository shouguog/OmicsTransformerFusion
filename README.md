# OmicsTransformerFusion
OmicsTransformerFusion integrates four omics modalities -- mRNA expression, miRNA expression, DNA methylation, and copy number variation -- from TCGA pan-cancer data to perform subtype identification. The framework combines modality-specific transformer encoders, cross-omics supervised contrastive learning, gated multi-modal fusion, and deep embedded clustering to predict cancer types. 
The analysis for JHS data from TopMed is in progress
